# ESTUDA-
O Estuda+ é um aplicativo educacional de microaprendizado gamificado voltado para estudantes do ensino médio, com foco no ensino de matemática, ciência e lógica. A proposta é transformar conteúdos tradicionalmente considerados difíceis em desafios curtos, interativos e acessíveis, estimulando o raciocínio lógico.
